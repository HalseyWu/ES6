# ES9

## rest参数

Spread 扩展运算符ES6中也使用过

在ES9中为对象提供了像数组一样的rest

```js
function connect(host, port, ...user) {
    console.log(host);
    console.log(port);
    console.log(user);
}
connect ({
    host: '127.0.0.1',
    port: 3306,
    username: 'root',
    password: 'root',
    type: 'master'
});
```

const F1 = {

## 正则扩展

```js
// 声明一个字符串
let str = '<a href="http://www.baidu.com">百度</a>';
// 提取url与标签文本
const reg = /<a href="(.*)">(.*)<\/a>/;
// 执行
const result = reg.exec(str);
console.log(result);
```

```js
let str = '<a href="http://www.baidu.com">百度</a>';
const reg = /<a href="(?<url>.*)">(?<text>.*)<\/a>/;
const result = reg.exec(str);
console.log(result);
console.log(result.group.url);
```
