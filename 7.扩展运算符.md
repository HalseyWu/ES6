# 扩展运算符

是一个符号 <kbd>...</kbd>扩展运算符可以将数组转换为逗号分隔的参数序列  

```js
// 声明一个数组
const tfboys = ["xi", "yuan", "kai"];       // "xi", "yuan", "kai"


// 声明一个函数
function yanchanghui () {
    console.log(arguments);
}
yanchanghui(tfboys);            // 是一个数组，但只有一个元素
                                // ["xi", "yuan", "kai"]

yanchanghui(...tfboys);         // yanchanghui("xi", "yuan", "kai") rest放在了形参...args 扩展在实参里面...
                                // 0: "xi"
                                // 1: "yuan"
                                // 2: "kai"
```


1. 数组合并
```js
const kuaizi = ["taili", "xiaoyang"];
const fenghuang = ["zengyi", "linghua"];
// const zuixuanapple = kuaizi.concat(fenghuang);       ES5之前提供了一个函数叫做concat
const zuixuanapple = [...kuaizi, ...fenghuang];     // ["", "", ""];
```


2. 数组克隆
```js
const sanzhihua = ["e", "g", "m"];
const sanyecao = [...sanzhihua];
console.log(sanyecao);
```


3. 将伪数组转为真正的数组
```html
...
<div></div>
<div></div>
<div></div>
...
```
```js
const divs = document.querySelectorAll("div");      // Type: NodeList
const divarr = [...divs];
console.log(divs);
console.log(divarr);        // 可以使用数组对应的方法 filters等 arguments也可以
```



## test
```js
const a = ["a", "b"];
const b = ["c", "d"];
const c = [..a, ...b];
const d = [...a];
console.log(c);             // "a", "b", "c", "d"
console.log(d);             // "a", "b"
```
