# 解构赋值

ES6 可以按照一定模式从数组和对象中提取值，并对变量进行赋值，这就被称为结构赋值

1. 数组的解构赋值
    ```js
    const F4 = ["张三", "李四", "王五", "赵六",];
    let [san, si, wu, liu] = F4;
    console.log(san);   // 张三
    console.log(si);    // 李四
    console.log(wu);    // 王五
    console.log(liu);   // 赵六
    ```


2. 对象的解构赋值
    ```js
    const zhang = {
        name: "张三三",
        age: 25,
        sing: function (){
            console.log("张三三正在唱歌")
        },
    };
    let { name, age, sing } = zhang;
    console.log(name);      // 张三三
    console.log(age);       // 25
    console.log(sing);      // ƒ () {
                            //     console.log("张三三正在唱歌")
                            // }
    sing();                 // 张三在唱歌
    ```
