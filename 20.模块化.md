# 模块化

## 引入

通过导入的形式

`<scrip type="module">`

```js
import * as m1 from "./src/js/m1.js"
console.log(m1);
```

解构赋值

```js
import {a, changge} from "./src/js/m1.js"
import {a as a, changge} from "./src/js/m1.js"
impoet {default as m1} from "./src/js/m1.js"
```

简便形式

```js
import m1 from "./src/js/m1.js"
```



```js
export let a = "清华";
export function changge() {
    console.log("我可以唱歌");
}
```

```js
let a = "清华";
function changge() {
    console.log("我可以唱歌");
}
export {a, changge};
```

```js
export default {
    a: "清华",
    changge: function() {
        console.log("我可以唱歌");
    }
}
```
