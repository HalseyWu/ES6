# 19.对象扩展

```js
console.log(Object.is(123, 123));
console.log(Object.is(NaN, NaN));
console.log(NaN === NaN);
```

> 注： 对于NaN来说他不等于任何值，包括他自己，但是在`object.is`里面他是相等的

## 对象的合并

```js
const config1 = {
    host: "localhost",
    root: "root",
    port: 3306,
    name: "root",
    password: "123456",
}
const config2 = {
    host: "http://baidu.com",
    root: "abc",
    port: 3307,
    name: "guangke",
    password2: "abcdef",
}
console.log(Object.assign(config1, config2));
```

## 设置原型对象

```js
const school = {
    name: "清华大学",
}
const cities = {
    xiaoqu: ['北京', '上海', '东莞']
}
Object.setPrototypeOf(school, cities);
console.log(Object.getPrototypeOf(school));
console.log(school);
```
