# 迭代器

遍历器(iterator)就是一种机制。它是一种接口，为各个不同的数据结构提供统一的访问机制。任何数据只要部署iterator接口，就可以完成遍历操作。

原生具备<kbd>iterator</kbd>接口的数据(可用<kbd>for...of...</kbd>遍历):

- Array
- Arguments
- Set
- Map
- String
- TypedArray
- NodeList

```js
const xiyouji = ["zhu", "xun", "tang", "sha"];
for(let v of xiyouji) {
    console.log(v);			// "zhu"
    						// "xun"
    						// "tang"
    						// "sha"
}
let iterator = xiyouji[Symbol.iterator]();
console.log(iterator.next());		// {value: "zhu", done: false}
console.log(iterator.next());		// {value: "xun", done: false}
console.log(iterator.next());		// {value: "tang", done: false}
console.log(iterator.next());		// {value: "sha", done: false}
console.log(iterator.next());		// {value: unfined, done: true}
```

## 部署iterator接口

```js
const BAN_JI = {
    name: "机应一班",
    studs: [
        "ming",
        "tian",
        "hui",
        "genghao"
    ],
    [Symbol.iterator]() {
        let index = 0;			// 定义索引
        let _this = this;		// 保存this
        return {
            next: function() {
                if(index < _this.studs.length) {
                    const RESULT = {
                        value: _this.studs[index],
                        done: false
                    };
                    index++;
                    return RESULT;
                } else {
                    return {
                        value: undefined,
                        done: true
                    };
                }
            }
        };
    }
};
for(let v of BAN_JI) {
    console.log(v);
}
```
