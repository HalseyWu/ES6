# 箭头函数

```js
    // let fn = function () {...}
    let fnn = (...) => {...}
```

1. this是静态的 this始终指向函数声明时所在作用域下的this值

```js
function getname () {
    console.log(this.name);
}

let getname2 = () => {
    console.log(this.name);
}

window.name = "广科";
const school = {
    name: "guangke";
};

// 常规函数调用
getname();              // 广科
getname2();             // 广科

// call方法
getname.call(school);           // guangke
getname2.call(school);          // 广科
```


2. 箭头函数不能作为构造函数实例化对象

```js
let person = (name, age) => {
    this.name = name;
    this.age = age;
}
let me = new person("lei", 25);
console.log(me);        // 报错: person不是一个构造函数
```


3. 不能使用arguments变量(特殊变量保存实参)

```js
let fn = () => {
    console.log(arguments);
}
fn(1, 2, 3);
```


4. 箭头函数简写
   1. 当形参有且只有一个时，可以省略小括号
   ```js
   let add = n => {
     return n+n;
   };
   console.log(add(9));         // 18
   ```

   2. 省略花括号 当代码体只有一句语句时可以省略，而且return必须省略且结果就是返回值
   ```js
   let pow = n => n*n;
   console.log(pow(9));         // 81
   ```
