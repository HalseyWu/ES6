

# dotAll模式

dot  .  元字符  除了换行符以外的任意单个字符

```js
// 字符串的引用有 '' "" ``
let str = `
<ul>
    <li>
        <a>泰坦尼克号</a>
        <p>上映日期: 2001-05-28</p>
    </li>
    <li>
        <a>肖生克的救赎</a>
        <p>上映日期: 2001-05-29</p>
    </li>
</ul>`;
// 正则创建
const egm = /<li>\s+<a>(.*?)<\/a>\s+<p>(.*?)<\/p>/
const result = egm.exec(str);
console.log(result);
```

```js
const egm = /<li>.*?<a>(.*?)<\/a>.*?<p>(.*?)<\/p>/gs;
let result;
let data = [];
while(result = egm.exec(str)) {
    data.push({title: result[1],
               time: result[2]});
}
```
