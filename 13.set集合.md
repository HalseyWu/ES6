# 13.set集合

**类似于数组，但成员的值都是唯一的**，集合实现了`iterator`接口，所以可以使用`扩展运算符`和`for…of…`进行遍历，集合的属性和方法:

- size返回集合的元素个数
- add增加一个新元素，返回当前集合
- delete删除元素，返回boolean值
- has检测集合中是否包含某个元素，返回boolean值
- clear清空集合，返回undefined

1. set里面的元素是唯一的

   ```js
   let s = new Set();
   console.log(s, typeof s);
   let s1 = new Set(["大事", "小事", "好事", "好事", "坏事"]);
   console.log(s1);		// "大事", "小事", "好事", "坏事"
   console.log(s1.size);	// 4
   ```

2. 添加元素

   ```sh
   s1.add("喜事");
   s1.delete("坏事");
   ```

3. 检测

   ```js
   console.log(s1.has("糟心事"));		// false
   ```

4. 清空

   ```js
   s1.clear();
   console.log(s1);		// undefined
   ```

5. 遍历

   ```js
   for(let v of s1) {
       console.log(v);
   }
   ```
   
## 实例

```js
let arr = [1, 2, 3, 4, 5, 4, 3, 2];
```

1. 数组去重

   ```js
   let result = [...new Set(arr)];
   ```

2. 求交集

   ```js
   let arr2 = [4, 5, 6, 7, 6, 5, 4];
   let result = [...new Set(arr)].filter(item => {
       let s2 = new Set(arr2);		4, 5
       if(s2.has(item)) {
           return true;
       } else {
           return false;
       }
   });
   console.log(result);
   ```
   
   简写
   ```js
   let result = [...new Set(arr)].filter(item => new Set(arr2).has(item))
   ```
   
3. 求差

   `arr - arr2`
   
   ```js
let result = [...new Set(arr)].filter(item => !(new Set(arr2).has(item)));
   ```

   `arr2 - arr`
   
   ```js
   let result2 = [...new Set(arr2)].filter(item => !(new Set(arr).has(item)));
   ```
   
4. 求并集

   ```js
   let union = [...new Set([...arr, ...arr2])];
   ```
