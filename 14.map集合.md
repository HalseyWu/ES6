# 14.map集合

类似于对象，也是键值对的集合。但“键”的范围不限于字符串，各种类型的值（包括对象）都可以当作键

```js
// 创建map数据结构
let m = new Map();
// 添加数据 set
m.set(["name", "易烊千玺"]);
m.set("changge", function() {
    console.log("是谁在唱歌");
});
let key = {
    school: "广科"
};
m.set(key, [1, 2, 3]);
```
