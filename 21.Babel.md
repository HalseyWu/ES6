# Babel

Babel能够将新的ES规范语法转换成ES5的语法

初始化项目

```sh
npm init -y
```

安装

```sh
npm i babel-cli babel-preset-env browserify
```

使用babel转换

```sh
npx babel <js目录> -d <转化后的js目录> --presets=babel-preset-env
```

打包

```sh
npx browserify <转化后的js目录> -o <打包后的目录>
```

引入bundle.js

```html
<script src="<bundle路径>" type="module"></script>
```
