# ES8

```js
// 声明一个对象
const school = {
    name: "清华",
    cities: ["北京", "东莞", "深圳"],
    xueke: ["ES6", "Vue", "webpack", "php"]
};
// 获取对象所有的键key
console.log(Object.keys(school));
// 获取对象所有的value
console.log(Object.values(school));
// entries
console.log(Object.entries(school));
const m = new Map(Object.ectries(school));
console.log(m.get("cities"));
```

对象属性的描述对象

```js
const obj = Object.create(null, {
    name: {
        // 设置值
        value: "清华",
        // 属性特征
        writable: true,
        configurable: true,
        enumerable: true
    }
});
console.log(Object.getOwnPropertyDescriptor(obj));
```
