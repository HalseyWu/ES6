# 18.数值扩展

```js
console.log(0.1 + 0.2)		// 0.30000000000004
```

## Number.EPSILON

表示最小精度

```js
function equal(a, b) {
    if(Math.abs(a - b) < Number.EPSILON) {
        return true;
    } else {
        return false;
    }
}

console.log(0.1 + 0.2 === 0.3);
console.log(equal(0.1 + 0.2, 0.3));
console.log(0.1 + 0.2 - 0.3);
```

## 进制转换

```js
// 0b开头的数字是二进制转十进制
let b = 0b1010;
// 0o开头的数字是八进制转十进制
let o = 0o777;
// 默认十进制
let d = 100;
// 0x开头的数字是十六进制转十进制
let x = 0xff;
console.log(b);
console.log(o);
console.log(d);
console.log(x);
```

## Number.isFinte()

检测一个数是否为有限的

```js
console.log(Number.isFinte(100));
console.log(Number.isFinte(100/0));
console.log(Number.isFinte(Infinity));
```

## Number.isNaN()

检测一个数是否为nan

```js
console.log(Number.isNaN(100));
console.log(Number.isNaN(100/0));
```

## Number.isInterger()

检测一个数是否为整数

```js
console.log(Number.isInterger(123.4));
```

## Number.parseInt() & Number.parseFloat()

用来将字符串转换为整数或浮点数

```js
console.log(Number.parseInt("1234abc"));
console.log(Number.parseFloat("1.234abc"));
```

## Math.trunc

用于去除小数部分保留整数部分

```js
console.log(Math.trunc(3.5));
```

## Math.sign

判断一个数为正数(1)、负数(-1)、零(0)

```js
console.log(Math.sign(200));
console.log(Math.sign(0));
console.log(Math.sign(-200));
```
