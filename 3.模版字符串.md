# 模版字符串

1. 字符串内容中可以直接出现换行
   ```js
   let str = `<ul>
                 <li>aaa</li>
                 <li>bbb</li>
                 <li>ccc</li>
                </ul>`;
   ```

2. 变量的拼接
   ```js
   let name = "吴";
   // let wo = name + "是个好学生";
   let wo = `${name}是个好学生`;
   console.log(wo);                     // 吴是个好学生
   ```
